# Arduino RC Filter Signal Analysis

This project studies the behavior of a first-order RC low-pass filter excited by a time-varying signal generated with Arduino PWM.

The experiment combines:

- Analog electronics (RC filter)
- Embedded systems (Arduino)
- Numerical modeling
- Signal analysis

## Circuit

The circuit is a standard RC low-pass filter:

R = 1kΩ  
C = 100µF  

Time constant:

τ = RC = 0.1 s

The input signal is generated by Arduino and follows:

V(t) = 2.5(1 + 0.5 sin(ω1 t) + 0.5 sin(ω2 t))

Two potentiometers control the angular frequencies ω1 and ω2.

The Arduino outputs a PWM signal whose duty cycle approximates this waveform.

## Measurements

The Arduino measures the voltage across the capacitor using the ADC and logs the following data:

- Input signal
- Measured capacitor voltage
- Theoretical RC response
- Potentiometer values
- Time

Data is exported as CSV files.

## Theoretical Model

The theoretical capacitor voltage is computed using the first-order RC exponential response:

Vc(t+Δt) = Vc(t) + (Vin − Vc(t))(1 − exp(−Δt/τ))

This allows comparison between the theoretical response and the real circuit.
